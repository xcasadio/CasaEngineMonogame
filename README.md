# CasaEngineMonogame
Editor + game engine written in C# .Net 6 and WPF with MonoGame.

## Objective:
I developed this engine with the aim of migrating all my game projects to the same engine.

## Features
* 2D
  - Sprites
  - Animations
  - 2d Map
* 3D
  - Models
  - Textures
* Physics
  - RigidBody

## 3rd parties
* Physics : BulletSharp
